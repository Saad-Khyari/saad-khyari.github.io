Simple Web Game: Guess the Number

This activity demonstrates a simple interactive game using HTML, CSS, and JavaScript. The player tries to guess a randomly generated number between 1 and 100. This project shows how to handle user input, DOM updates, and basic logic.

How it Works

The game generates a random number. The player enters a guess, and the game responds with hints: "Too high", "Too low", or "Correct!". This demonstrates basic game logic and interactivity in JavaScript.
